# Notas 
App para crear tablas de multiplicar en "txt"

```
Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     La base de la tabla de multiplicar        [number] [default: 3]
  -l, --listar   Muestra la tabla en consola          [boolean] [default: false]
  -h, --hasta    El numero hasta cuanto queremos multiplicar
                                                          [number] [default: 10]

```
